# WCSAC_with_discrete_action_space
Pytorch-based the discrete version of WCSAC

- Python *3.8.19*
- You need to install the *[safety_gymnasium](https://github.com/PKU-Alignment/safety-gymnasium)*

Then, how to run it, as follows:

```shell
./main_WCSAC.py
```
